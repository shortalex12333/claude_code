# LinkedIn Content Workflow & Approval Process
**Program:** LKD-MKT-2025-Q4
**Purpose:** Standardized workflow for content creation, review, and publishing
**Time Investment:** 2-3 hours/week for full execution

---

## WORKFLOW OVERVIEW

**Total Weekly Time:** ~2-3 hours
- Planning: 30 minutes (Monday)
- Creation: 60-90 minutes (spread across week)
- Engagement: 30-45 minutes (daily, 5-10 min increments)
- Review: 15 minutes (Friday)

**Principle:** Batch creation, daily engagement, weekly review

## CONTENT QUALITY CHECKLIST

**Use before scheduling ANY post:**

### ‚úÖ Strategic Alignment
- [ ] Fits clearly into one content pillar
- [ ] Targets specific persona (not generic)
- [ ] Advances business goal (awareness/consideration/decision)
- [ ] Aligns with brand voice

### ‚úÖ Hook Quality
- [ ] First 120 characters create curiosity
- [ ] Opens with specific statement (not vague)
- [ ] Uses hook formula from library
- [ ] Promise of value clear

### ‚úÖ Structure
- [ ] Follows archetype template
- [ ] Logical flow (hook ‚Üí body ‚Üí CTA)
- [ ] Scannable (line breaks, formatting)
- [ ] One clear idea (not multiple topics)

### ‚úÖ Value Delivery
- [ ] Tactical, emotional, OR intellectual value
- [ ] Specific (numbers, names, examples)
- [ ] Backed by evidence (data, experience, logic)
- [ ] Not generic advice everyone shares

### ‚úÖ Algorithm Optimization
- [ ] 800-1200 character count
- [ ] No external links in main post
- [ ] Line breaks every 1-2 sentences
- [ ] 2-3 specific hashtags (not generic)
- [ ] CTA drives engagement (question/invitation)

### ‚úÖ Engagement Design
- [ ] CTA is specific (not "what do you think?")
- [ ] Easy to respond to (low friction)
- [ ] Invites discussion or answers
- [ ] Will commit to replying quickly

### ‚úÖ Brand Safety
- [ ] No controversial topics outside expertise
- [ ] No engagement bait ("comment YES")
- [ ] No unverified claims
- [ ] No negative competitor mentions
- [ ] Grammar and spelling correct

**Scoring:**
- 20-21 checks: Excellent, publish with confidence
- 18-19 checks: Good, minor tweaks needed
- 15-17 checks: Needs significant revision
- <15 checks: Start over with template

## APPROVAL PROCESS

### Solo Creator (No Approval Needed)
**You own:** Personal page content
**Process:** Use quality checklist, self-approve
**Risk:** Low (personal brand, can iterate)

### Small Team (Peer Review)
**You own:** Personal + company page
**Process:**
1. Creator drafts post
2. Peer reviews against checklist
3. Creator implements feedback
4. Both sign off
**Timeline:** 24-hour turnaround

### Larger Org (Formal Approval)
**You own:** Company page + exec personal pages
**Process:**
1. Creator drafts post using templates
2. Manager reviews against checklist (24 hours)
3. Legal reviews if needed (48 hours for sensitive topics)
4. Final approval from content owner
5. Schedule publication
**Timeline:** 2-5 days (plan ahead!)

## EMERGENCY CONTENT PROTOCOL

**When scheduled content must be replaced:**

### Scenario 1: Breaking News in Industry
**Decision:** Swap planned post for timely commentary

**Process (30-minute emergency workflow):**
1. Assess: Is this relevant to our audience? (Y/N)
2. If YES: Does this fit a pillar? (Force to P2 or P3 if needed)
3. Use Opinion Post template (fastest to write)
4. Keep to 600-800 characters (speed over depth)
5. Post within 2-4 hours of news breaking
6. Engage aggressively for first hour

**Guardrail:** Only do if:
- Directly affects your audience
- You have informed perspective
- Can post within 4 hours (after that, not timely)

---

### Scenario 2: Post Mistake (Typo, Wrong Info)
**Decision:** Edit immediately if <5 minutes, otherwise leave

**Process:**
1. If <5 minutes since publish: Edit post, add comment noting correction
2. If >5 minutes: Leave post, add comment with correction
3. If factually wrong: Delete, post correction, apologize
4. Document in tracking sheet

**Guardrail:** Own mistakes quickly and transparently

---

### Scenario 3: Negative Backlash
**Decision:** Assess severity, respond proportionally

**Process:**
1. **Minor criticism (disagrees):** Respond professionally, invite discussion
2. **Valid criticism (you're wrong):** Acknowledge, thank, commit to learning
3. **Troll/Harassment:** Do not engage, block if necessary
4. **Major backlash:** Assess if post should be deleted (rare, save face)

**Guardrail:** Never get defensive, always professional

## REPURPOSING & RECYCLING

**Principle:** Great content deserves second life

### What to Repurpose:
- Posts with >6% engagement rate
- Posts with >20 saves
- Posts with deep comment discussions
- Posts that drove DMs or leads

### How to Repurpose:
1. **Repost (3-6 months later)**
   - Same content, updated data
   - Add: "Reposting because this got 500 saves last time:"

2. **Expand into Thread**
   - Turn single post into 5-7 connected posts
   - Post over one week, reference each other

3. **Turn into Carousel**
   - Take text post, visualize in slides
   - Different format = different audience reach

4. **Create Video Version**
   - Record yourself explaining concept
   - 60-90 seconds, authentic phone video

5. **External Content**
   - Blog post (expand 3-5x)
   - Newsletter (curate 5-10 related posts)
   - Webinar (deep dive on topic)

**Frequency:** Repurpose top 10% of content quarterly

## WORKFLOW TROUBLESHOOTING

### Problem: "I don't have time to create content"
**Solution:**
- Batch creation: 90 minutes = 3 weeks of content
- Use templates religiously (don't start from scratch)
- Reduce posting frequency (3x/week vs 5x)
- Repurpose more aggressively

### Problem: "I don't know what to write"
**Solution:**
- Open Hook Library, pick any hook
- Open calendar, topic is already chosen
- Use archetype template (structure given)
- Write about customer conversation from today

### Problem: "My posts aren't getting engagement"
**Solution:**
- Run A/B tests (see AB_Testing_Protocol.md)
- Engage more with others (algorithm rewards reciprocity)
- Post at optimal times (Tuesday 8:30am)
- Make CTA more specific (not "thoughts?")

### Problem: "I'm running out of ideas"
**Solution:**
- Repurpose top posts (update and reshare)
- Answer comment questions as posts
- Share customer stories (with permission)
- Commentary on industry news
